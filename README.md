# 用模拟退火算法估计Heston期权定价模型参数

## 项目描述

Heston期权定价模型在进行期权定价时，需要填入五个已知参数。这五个参数需要达到定价误差最小，因此这本质上是一个误差极小化问题。本项目使用模拟退火算法来估计Heston模型的五个参数。

## 资源内容

2. **期权数据**：用于模型训练和验证的期权数据。

## 使用说明

1. **解压文件**：首先解压下载的压缩包。
